Доработай работу с клиентами в CRM. Сейчас при создании заказа и в карточке клиента не хватает нормальной связки customer → orders. Нужно сделать это удобно для менеджеров.

1. ПРИ СОЗДАНИИ ЗАКАЗА АРЕНДЫ НУЖНО АВТОПРЕДЛОЖЕНИЕ КЛИЕНТА
==================================================

Сейчас при создании заказа аренды нужно сделать полноценный autocomplete клиента.

Нужно:
- автопредложение клиента при вводе
- поиск по:
  - имени
  - телефону
  - email
- особенно важно: при вводе почты тоже должен находиться существующий клиент
- если клиент найден, данные должны подставляться автоматически
- если клиент не найден, должна быть возможность быстро создать нового

То есть менеджер начинает вводить:
- email
или
- телефон
или
- имя

и система предлагает совпадения из customers.

==================================================
2. ПРИ ВЫБОРЕ КЛИЕНТА ДОЛЖНО БЫТЬ АВТОЗАПОЛНЕНИЕ
==================================================

Если клиент найден и выбран, форма заказа должна автоматически подставлять:

- имя
- телефон
- email
- preferred communication channel
- заметки клиента, если они есть
- при необходимости адрес / комментарии, если это поддерживается

Важно:
- не просто выбрать customerId
- а реально заполнить customer block в форме
- чтобы менеджер сразу видел, кого оформляет

==================================================

В customer details page нужно добавить быстрые действия.

В карточке клиента должны быть кнопки:

- Создать заказ аренды
- Создать заказ продажи
- Создать бронирование тура, если туры уже встроены в flow
- Открыть чат, если есть chat session
- Редактировать клиента

Логика:
- если менеджер нажал “Создать заказ аренды” из карточки клиента

  - этот клиент уже подставлен автоматически
- если нажал “Создать заказ продажи”
  - открывается create sale order
  - клиент уже подставлен
