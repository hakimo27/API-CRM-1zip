28. КОРЗИНА И ПОКУПКА НЕ РАБОТАЮТ
==================================================

Сейчас корзина и сам flow покупки/оформления заказа работают не до конца или не работают вообще. Нужно считать это критичной проблемой.

Проблемы:
- корзина не работает как нужно
- нет понятного действия “Купить”
- непонятно, как товар уходит в заказ
- sale flow и rental flow могут быть частично сломаны
- пользователь не должен попадать в тупик

Нужно провести аудит и исправить это системно.

### Что проверить
Проверить отдельно:

#### Для продажи
- add to cart на карточке sale product
- add to cart в списке sale products
- cart drawer / cart page
- update quantity in cart
- remove item from cart
- cart total calculation
- checkout sale
- create sale order
- success page / confirmation
- CTA buttons:
  - Купить
  - В корзину
  - Узнать о наличии
  - Оставить заявку

#### Для аренды
- add to rental cart
- rental cart page
- update quantity
- change dates
- availability recheck
- checkout rental
- create rental order
- success flow

### Для продажи должно быть понятно
На sale pages должны быть корректные CTA:
- Купить
- В корзину
- Узнать о наличии
- Оставить заявку

Не должно быть:
- “Взять в аренду” на странице продажи
- пустых кнопок
- кнопок без действия
- товара без пути в заказ

### Что нужно сделать
1. Найти root cause, почему корзина не работает
2. Проверить cart state management
3. Проверить local storage / session cart
4. Проверить add-to-cart actions
5. Проверить cart API / checkout API
6. Проверить total recalculation
7. Проверить checkout flow
8. Проверить success flow после оформления

### Важно
- если товар sale, то пользователь должен иметь понятный путь:
  карточка → купить / в корзину → корзина → checkout → заказ
- если товар rental, то:
  карточка → выбрать даты/количество → в корзину → checkout → заказ
- не должно быть “все есть, но купить нельзя”
